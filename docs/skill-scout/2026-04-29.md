## 1. Proactive Agent 🦞（主动型 Agent 框架）

**链接：** https://clawhub.ai/halthelobster/proactive-agent

**功能描述：**
由 Hal Labs 出品的 Agent 进化框架，v3.1.0 最新版。核心特性包括：WAL 协议（先写后答，防止细节丢失）、Working Buffer（上下文危险区日志记录）、Compaction Recovery（上下文压缩后自动恢复）、自主 Cron 任务（isolated agentTurn 模式）、实现验证（区分"文字改了"和"行为改了"）、10次尝试后才寻求帮助的"顽强执行力"。下载量 150k，⭐ 734。

**适用场景：**
- 希望 Agent 主动发现需求、而不是等待指令
- 长会话、多 session 场景下防止上下文丢失
- 构建有自我改进能力的定时维护 Agent
- 开发自动化 cron 工作流时区分"提示型"和"执行型"任务

## 2. AdMapix（广告情报 & 应用市场分析）

**链接：** https://clawhub.ai/fly0pants/admapix

**功能描述：**
接入 AdMapix 广告情报平台，支持搜索竞品广告素材、分析 App 投放策略、查看应用排行榜、追踪下载量/收入趋势、深度市场分析。支持中英文双语，自动切换输出格式。内置深度研究框架（Deep Research），对于"分析 Temu 投放策略"等复杂任务，可自动编排多个 API 调用并生成结构化报告。下载量 127k，⭐ 273。

**适用场景：**
- 竞品广告素材监控与创意灵感收集
- 游戏/电商 App 全球投放渠道分析
- 应用商店排行榜追踪与市场趋势研究
- 出海团队的广告策略辅助决策

## 3. PollyReach 🦜（AI 电话助理）

**链接：** https://clawhub.ai/pollyreach/pollyreach

**功能描述：**
给 AI Agent 分配真实电话号码，让 Agent 代你打电话办事。支持：外呼（订餐、订酒店、投诉、询价、批量面试约谈等）、来电接听（会议中自动接听并推送摘要）、来电过滤（屏蔽骚扰电话）、自定义接听提示词（设定 Agent 的接听角色）。下载量 96.2k，⭐ 37，仍在活跃增长。

**适用场景：**
- 让 Agent 帮你预约餐厅、酒店、政府服务
- 出差海外时跨语言打电话办事
- 作为企业 24×7 AI 前台接待来电
- 会议中拦截来电并自动汇总

## 4. Obsidian（Obsidian 笔记自动化）

**链接：** https://clawhub.ai/steipete/obsidian

**功能描述：**
通过 obsidian-cli 操控 Obsidian Vault（普通 Markdown 文件夹），支持搜索笔记/全文检索、创建笔记、移动/重命名（自动更新所有 wikilinks 引用）、删除笔记。不依赖 Obsidian 插件 API，直接操作文件系统，与 Obsidian 实时同步。今日（2026-04-29）刚更新，下载量 87.4k，⭐ 360。

**适用场景：**
- 让 Agent 自动整理、归档笔记
- 根据对话内容自动创建知识条目
- 批量重命名/移动笔记并保持链接完整性
- 将 Obsidian 作为 Agent 的个人知识库

## 5. Skill Vetter 🔒（Skill 安全审核）

**链接：** https://clawhub.ai/spclaudehome/skill-vetter

**功能描述：**
安装任何外部 Skill 前的安全审核协议。对 Skill 进行四步检查：来源可信度评估、代码红旗扫描（检测 curl/wget、数据外传、凭据读取、base64 解码、eval 执行等危险模式）、权限范围评估、风险等级分类（🟢 低/🟡 中/🔴 高/⛔ 极高），最终输出结构化审核报告。研究显示约 26% 的社区 Skill 存在安全漏洞。下载量 225k，⭐ 984。

**适用场景：**
- 从 ClawHub、GitHub 安装第三方 Skill 前的安全把关
- 评估 Skill 权限边界是否合理
- 团队统一的 Skill 安全引入流程
- 识别潜在的提示注入或数据外泄风险
