# ⭐ 今日 Skill 推荐 — 2026-05-01

本期精选 5 个 ClawHub 热门 Skill，覆盖 GitHub 工作流、天气查询、无头浏览器自动化、PDF 自然语言编辑、自动更新维护等实用场景。

## 1. Github（GitHub CLI 集成）

**链接：** https://clawhub.ai/steipete/github

**功能描述：**
通过 `gh` CLI 与 GitHub 深度集成的 Skill，支持 `gh issue`、`gh pr`、`gh run`、`gh api` 等完整命令集，覆盖 Issue 管理、PR 审查、CI 运行追踪、高级 API 查询等场景。下载量 168k，⭐ 550，由 @steipete 维护，持续活跃更新。

**适用场景：**
- 快速查看和管理 GitHub Issue / PR 状态
- 监控 CI/CD 流水线运行结果
- 结合 coding-agent 实现"写代码 → 提 PR → 审查"全链路自动化
- 使用 `gh api` 进行高级批量查询与操作

## 2. Weather（免 API Key 天气查询）

**链接：** https://clawhub.ai/steipete/weather

**功能描述：**
无需任何 API Key 即可获取当前天气与未来预报的轻量 Skill。基于公开天气数据源，支持全球任意城市查询，返回温度、湿度、风速、天气状况等详细信息。下载量 143k，⭐ 372，开箱即用。

**适用场景：**
- 快速查询任意城市当前天气
- 出行、活动策划前获取天气预报
- 嵌入 Agent 工作流自动判断天气条件（如"明天下雨就提醒带伞"）
- 零配置部署，适合新手快速体验

## 3. Agent Browser（AI 优化的无头浏览器自动化）

**链接：** https://clawhub.ai/matrixy/agent-browser-clawdbot

**功能描述：**
专为 AI Agent 优化的无头浏览器自动化 CLI。核心亮点是**可访问性树快照（Accessibility Tree Snapshot）**+ **ref 引用式元素选择**，让 Agent 能像人一样"看懂"网页结构并精准操作，而无需依赖脆弱的 CSS 选择器。下载量 101k，⭐ 365。

**适用场景：**
- Agent 驱动的网页数据抓取与动态内容提取
- 自动化填写表单、点击按钮、登录操作
- 无头模式运行，适合服务器/CI 环境
- 相比 Playwright MCP，更适合轻量级 Agent 集成

---

## 4. Nano Pdf（自然语言驱动的 PDF 编辑）

**链接：** https://clawhub.ai/steipete/nano-pdf

**功能描述：**
通过 `nano-pdf` CLI，让 Agent 用自然语言指令编辑 PDF 文件，无需了解复杂 PDF 规范。支持内容提取、文本修改、页面操作等常见 PDF 编辑任务，大幅降低 PDF 处理门槛。下载量 97.5k，⭐ 235，@steipete 出品。

**适用场景：**
- 提取合同、报告中的关键信息
- 批量修改 PDF 文档内容
- 合并、拆分、重排 PDF 页面
- 让 Agent 自动处理收到的 PDF 附件

## 5. Auto-Updater Skill（每日自动更新维护）

**链接：** https://clawhub.ai/maximeprades/auto-updater

**功能描述：**
每日自动检查并更新 OpenClaw 及所有已安装 Skill，完成后向用户推送"哪些 Skill 更新了、改了什么"的变更摘要。通过 cron 定时运行，无需手动维护，保持 Agent 能力始终最新。下载量 79.6k，⭐ 388，@maximeprades 维护。

**适用场景：**
- 懒得手动更新 Skill 的用户，实现零维护
- 团队共享 Agent 环境统一自动保持最新版本
- 配合 Skill Vetter 使用：先自动安全审核，再自动更新
- 适合生产环境中长期运行的 Agent 系统
