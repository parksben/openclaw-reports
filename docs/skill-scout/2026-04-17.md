## 1. Elite Longterm Memory

**功能描述**

为 OpenClaw Agent 提供持久化长期记忆系统，基于 WAL（Write-Ahead Log）协议 + 向量搜索 + git-notes 三层架构。支持本地 LanceDB 向量存储，可选接入 Mem0、SuperMemory 等云端记忆服务。

**适用场景**

- 需要跨会话记住用户偏好、项目上下文的 Agent
- 构建有"记忆"的个人助理或工作流 Agent
- 希望在上下文压缩后仍能恢复关键信息的场景

**安装**

```bash
clawhub install elite-longterm-memory
```

**链接：** https://clawhub.ai/nextfrontierbuilds/elite-longterm-memory

## 2. Web Scraping

**功能描述**

指令型网页抓取助手，封装了 `web_fetch`、`browser`、`web_search` 三种抓取策略的最佳实践。涵盖分页处理、去重、截图、登录态页面交互等场景，内置限速建议防止被封。

**适用场景**

- 批量抓取公开数据（新闻、价格、招聘信息等）
- 需要处理 JS 渲染页面时切换 browser 模式
- 构建数据采集 Pipeline 的起点

**安装**

```bash
clawhub install web-scraping
```

**链接：** https://clawhub.ai/zhangqixin9527/web-scraping

## 3. OpenClaw GitHub Assistant

**功能描述**

GitHub 全功能操作助手，通过 Personal Access Token 直接调用 GitHub API。支持列出仓库、查看 CI 状态、管理 Issues/PRs、创建仓库、搜索代码、查看提交记录等操作。安全评级：Benign（高置信度）。

**适用场景**

- 日常 GitHub 工作流自动化（PR review 提醒、Issue 跟踪）
- CI/CD 状态监控与通知
- 多仓库批量操作脚本化

**安装**

```bash
clawhub install openclaw-github-assistant
```

**链接：** https://clawhub.ai/conorkenn/openclaw-github-assistant

## 5. Code Review (quack-code-review)

**功能描述**

基于 LogicArt 远程分析引擎的代码审查 Skill，将代码提交至 `https://logic.art/api/agent/analyze` 进行智能分析，返回问题报告和改进建议。适合快速获取代码质量反馈。

**注意：** 代码会发送至第三方服务，提交前请移除敏感信息（密钥、密码等）。

**适用场景**

- PR 合并前的快速代码质量检查
- 重构前的问题摸底
- 团队代码规范执行辅助

**安装**

```bash
clawhub install quack-code-review
```

**链接：** https://clawhub.ai/jpaulgrayson/quack-code-review
