## 1. gh-issues — GitHub Issue 自动修复机器人

**作者：** jackhua6  
**链接：** https://clawhub.ai/jackhua6/gh-issues

### 功能描述

全自动 GitHub Issue 处理流水线。给定一个仓库，自动拉取 Issue 列表、分析可行性、并行派生子 Agent 修复代码、推送分支、开 PR，最后汇报结果。支持 fork 模式、watch 轮询、cron 定时运行，以及 PR Review 评论自动响应。

### 核心能力

- 并行最多 8 个子 Agent 同时修复不同 Issue
- 自动跳过已有 PR 或正在处理的 Issue（防重复）
- `--cron` 模式：每次只处理一个 Issue，适合定时任务
- `--reviews-only`：专门处理 PR Review 评论，自动 push 修复
- 支持 `--notify-channel` 将结果推送到 Telegram

### 适用场景

- 开源项目维护者批量处理 bug 类 Issue
- 团队自动化 CI/CD 流程，减少人工干预
- 定期扫描仓库积压 Issue 并自动尝试修复

## 2. skill-creator — Skill 创作向导

**作者：** chindden  
**链接：** https://clawhub.ai/chindden/skill-creator

### 功能描述

帮助你从零创建高质量 OpenClaw Skill 的元技能。提供完整的 Skill 设计方法论：如何规划 scripts/references/assets 目录结构、如何写好 SKILL.md frontmatter 触发描述、如何用渐进式披露设计控制 context 消耗。

### 核心能力

- 引导式需求收集，通过具体示例理解 Skill 用途
- 自动初始化 Skill 目录结构（`init_skill.py`）
- 提供 3 种 Progressive Disclosure 设计模式
- 打包验证（`package_skill.py`）确保 Skill 符合规范

### 适用场景

- 想把重复性工作流封装成可复用 Skill
- 为团队创建内部知识库 Skill（含 API 文档、业务逻辑）
- 改进现有 Skill，减少 context 浪费

## 3. github — GitHub CLI 操作助手

**作者：** steipete  
**链接：** https://clawhub.ai/steipete/github

### 功能描述

基于 `gh` CLI 的 GitHub 操作指南。覆盖 PR 状态检查、CI 运行查看、Issue 管理、JSON 结构化输出等常用场景，提供即用型命令示例。

### 核心能力

- `gh pr checks` 查看 PR CI 状态
- `gh run view --log-failed` 快速定位失败步骤
- `gh api` 访问 GitHub REST API 高级查询
- `--json` + `--jq` 组合实现结构化数据过滤

### 适用场景

- 日常开发中快速查 PR/Issue 状态
- 在 Agent 工作流中集成 GitHub 操作
- 不想记 gh 命令参数时的速查手册

## 4. weather — 免费天气查询

**作者：** steipete  
**链接：** https://clawhub.ai/steipete/weather

### 功能描述

无需 API Key 的天气查询 Skill，整合 wttr.in 和 Open-Meteo 两个免费服务。支持城市名、机场代码、坐标查询，可输出简洁单行格式或完整三日预报。

### 核心能力

- `wttr.in` 一行命令获取当前天气（含 emoji 图标）
- 支持自定义格式码：温度、湿度、风速、月相
- `Open-Meteo` 作为备用，返回 JSON 适合程序化处理
- 可保存天气图片到 `/tmp/weather.png`

### 适用场景

- 早报/日报 Agent 自动附加当日天气
- 出行提醒场景（结合日历 Skill）
- 需要天气数据但不想申请 API Key 的快速集成

## 5. gemini — Gemini CLI 集成

**作者：** steipete  
**链接：** https://clawhub.ai/steipete/gemini

### 功能描述

将 Google Gemini CLI 接入 OpenClaw Agent 工作流的 Skill。适合需要调用 Gemini 做一次性问答、内容生成或摘要的场景，作为现有模型的补充或对比验证工具。

### 核心能力

- 调用本地 `gemini` 二进制执行推理任务
- 支持交互式登录授权流程
- 可与其他 Skill 组合，实现多模型协作
- 注意：需要本地安装 `gemini-cli`（`brew install gemini-cli`）

### 适用场景

- 需要 Gemini 长上下文能力处理大文档
- 对比不同模型输出结果
- 在 Agent 流程中按需切换模型
