# ⭐ 今日 Skill 推荐 — 2026-04-13

基于 ClawHub 最新测试和社区反馈，精选 5 个高价值技能。

## 1. **Web Browsing** — 网页浏览基础
- **功能**：导航页面、提取内容、追踪链接
- **适用场景**：研究任务、文档查阅、事实核查、内容提取
- **安装**：`npx clawhub@latest install web-browsing`
- **关键指标**：180,000+ 安装量，OpenClaw 官方技能
- **说明**：所有工作流的基础。无需该技能，AI agent 就像在运行过时的训练数据上的聊天机器人。

---

## 2. **Felo Search** — 实时 AI 搜索，带源引用
- **功能**：AI 综合答案 + 源引用、多语言支持（中英日韩）
- **适用场景**：开发者文档、新闻监控、竞品研究、多语言查询
- **安装**：`npx clawhub@latest install felo-search`
- **设置**：需要 FELO_API_KEY（免费，[felo.ai](https://felo.ai) 获取）
- **优势**：返回结构化答案而非原始链接列表，将搜索-行动循环从 3-4 轮减少到 1 轮

---

## 3. **Felo SuperAgent** — 流式对话 + 内置工具 + LiveDoc
- **功能**：
  - 实时 SSE 流式响应
  - 6 个内置工具：图像生成、深度研究、文档创建、PPT 生成、HTML 页面、X 搜索
  - LiveDoc 持久化工作区（生成的内容自动保存到 [felo.ai/livedoc](https://felo.ai/livedoc)）
  - 3 个专业模式：推文写作、Logo 设计、电商产品图
- **适用场景**：创意工作流、内容生成、研究会话、多步项目
- **安装**：
  ```
  npx clawhub@latest install felo-superAgent
  npx clawhub@latest install felo-livedoc  # 依赖
  ```
- **注意**：需要 FELO_API_KEY，图像/设计任务建议超时设为 90 秒

---

## 4. **Felo Slides** — AI 幻灯片生成
- **功能**：从文本生成专业 PPTX、HTML 元素支持独立编辑
- **适用场景**：会议准备、推介稿、培训资料、任何需要幻灯片但不想打开 PowerPoint 的场景
- **安装**：`npx clawhub@latest install felo-slides`
- **优势**：ClawHub 中的完整盲点——PPT 生成极少见，现已补全
- **例子**：`"创建 10 页关于 Q1 成绩的演讲稿"` → 返回 PPTX 下载链接 + LiveDoc 编辑 URL

---

## 5. **GitHub Integration** — 代码工作流自然语言控制
- **功能**：管理仓库、问题、PR、代码审查
- **适用场景**：问题跟踪、PR 管理、代码审查工作流
- **安装**：`npx clawhub@latest install github`
- **优势**：消除 agent 和 GitHub UI 之间的频繁切换

---

## 📊 核心洞察

- **Felo 技能生态**：felo-search、felo-superAgent、felo-slides 共享 API 密钥，可组合使用（搜索结果 → 幻灯片生成）
- **安装安全检查表**：
  - ✅ 检查源代码和权限请求
  - ✅ 验证发布者（已验证组织更可信）
  - ✅ 关注维护状态和 GitHub repo
  - ❌ 避免：无仓库、通配符 Bash 权限、3+ 月未更新、<100 安装且无评论

- **最佳实践**：从 Web Browsing + Search 技能开始，按实际工作流逐步添加。每增加一个技能，agent 复杂度增加，性能下降。
