# ⭐ 今日 Skill 推荐 — 2026-04-14

精选 ClawHub 热门 5 大 skills，每一个都是经过实测验证、高下载量且安全可靠的工具。

---

## 1. **Web Browsing** — 网页浏览与内容提取

**功能描述：** 为 OpenClaw 代理提供实时网页访问能力，能够导航页面、提取内容、跟踪链接。

**适用场景：** 
- 研究报告与文档查阅
- 事实核查
- 内容提取与分析

**技术信息：**
- 安装命令：`npx clawhub@latest install web-browsing`
- 下载量：180,000+ 次
- 状态：官方核心 skill，广泛应用基础

**为什么推荐：** 没有网页访问能力，你的 AI 代理就是个运行在旧数据上的聊天机器人。这是几乎所有高级工作流的基础依赖。

---

## 2. **Felo Search** — AI 驱动的实时搜索与源引用

**功能描述：** 不返回链接列表，而是直接返回 AI 综合的答案并附带源引用。支持中英日韩多语言。

**适用场景：**
- 开发者文档查询
- 新闻监控
- 竞争力分析
- 多语言跨地区查询

**技术信息：**
- 安装命令：`npx clawhub@latest install felo-search`
- 需要 FELO_API_KEY（felo.ai 免费获取）
- 多语言原生支持（CJK + 英文）

**为什么推荐：** 相比传统搜索工具返回 10 个蓝色链接，Felo Search 将 3-4 轮 AI 对话压缩到 1 轮。如果你操作多语言工作流，这是目前最稳定的选择。

---

## 3. **Felo SuperAgent** — 多工具创意工作室

**功能描述：** 集实时流式对话、LiveDoc 持久化工作区、6 大内置工具于一体。支持实时 SSE 流式响应，每次对话生成的内容自动保存到可共享的 canvas。

**内置工具：**
- 图像生成
- 深度研究报告生成
- 文档创建
- PPT 幻灯片生成
- HTML 页面生成
- X/Twitter 搜索

**适用场景：**
- 创意内容生成（营销文案、视觉设计）
- 研究报告编写
- 演示幻灯片制作
- 多步骤项目协作

**技术信息：**
- 安装命令：`npx clawhub@latest install felo-superAgent && npx clawhub@latest install felo-livedoc`
- 需要 FELO_API_KEY
- 推荐超时设置：90 秒（图像/设计任务）

**为什么推荐：** ClawHub 上唯一整合了流式对话 + 持久 canvas + 资产生成的 skill。大多数竞品只能输出文本，这个把实际成果保存下来。

**适用场景：**

- 团队通知推送
- 移动端优先工作流

**技术信息：**
- 安装命令：`npx clawhub@latest install telegram`
- 下载量：145,000+ 次
- 设置时间：~5 分钟

**为什么推荐：** 简单但强大。提升生产力的最直接方式——把 AI 代理带到你口袋里。

## 5. **Felo Slides** — AI 演示文稿生成

**功能描述：** 通过自然语言描述自动生成专业 PPTX 演示文稿。输出的幻灯片采用 HTML 渲染，每个元素独立可编辑，还包含 LiveDoc 编辑链接。

**适用场景：**
- 季度财报演示
- 客户提案文稿
- 内部培训材料
- 会议演讲准备

**技术信息：**
- 安装命令：`npx clawhub@latest install felo-slides`
- 需要 FELO_API_KEY
- 输出格式：可编辑 PPTX + LiveDoc URL

**为什么推荐：** ClawHub 上唯一专业级别的演示文稿生成工具。绝大多数推荐文章都忽视了这个空白，现在有了。

## 安装所有 5 个 Skill

```bash
# 基础工具
npx clawhub@latest install web-browsing
npx clawhub@latest install telegram

# Felo 系列（需要统一 API Key）
npx clawhub@latest install felo-search
npx clawhub@latest install felo-superAgent
npx clawhub@latest install felo-livedoc
npx clawhub@latest install felo-slides
```

获取 FELO_API_KEY：访问 [felo.ai](https://felo.ai) → Settings → API Keys（当前开放免费期）
