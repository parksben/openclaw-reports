## 1. Ontology（类型化知识图谱 Agent 记忆）

**链接：** https://clawhub.ai/oswalpalash/ontology

**功能描述：**
@oswalpalash 出品，下载量 176k，⭐ 585，今日刚更新。核心理念：**用类型约束的知识图谱为 Agent 构建可验证的结构化记忆**。所有信息以 Entity（实体）+Relation（关系）形式存储，支持内置类型：Person、Organization、Project、Task、Goal、Event、Document、Note、Message 等。每次写入前先校验类型约束，防止脏数据。存储为 `memory/ontology/graph.jsonl` 追加日志，支持 query / relate / traverse 等图操作，复杂图可迁移 SQLite。

**适用场景：**
- 对话中说"记住 Alice 负责 Website Redesign 项目"，自动建立 Person → Project 关联
- 查询"X 依赖哪些任务"时执行图遍历而非模糊搜索
- 多 Skill 共享结构化状态（如 Task 状态在不同 Skill 间同步）
- 构建 Agent 个人 CRM：联系人、项目、会议、文档全部图化
- 配合 Memory Tiering Skill 使用，HOT 层存活跃节点，COLD 层归档历史图

## 2. Gog（Google 全家桶 CLI 集成）

**链接：** https://clawhub.ai/steipete/gog

**功能描述：**
@steipete 出品，下载量 168k，⭐ 877，v1.0.0。通过 `gog` CLI 统一操控 Google Workspace 全套服务：Gmail（搜索/发送/回复）、Calendar（查询/创建事件）、Drive（搜索文件）、Contacts（联系人管理）、Sheets（读写单元格/追加/清除）、Docs（导出/查看）。一次 OAuth 授权即可接入全部服务，支持 `--json` 输出方便脚本集成，环境变量 `GOG_ACCOUNT` 避免重复指定账号。

**适用场景：**
- 每天早上自动汇总今日 Gmail 未读邮件 + Google Calendar 日程
- 让 Agent 直接读写 Google Sheets 数据，替代人工维护表格
- 会议结束后自动生成会议纪要并存入 Google Docs
- 定时备份 Drive 文件列表，监控重要文件变更
- 与 ClickUp/Notion Skill 联动：同步 Google Calendar 事件到任务系统

## 3. Multi Search Engine（16 引擎中英双轨聚合搜索）

**链接：** https://clawhub.ai/gpyangyoujun/multi-search-engine

**功能描述：**
@gpyangyoujun 出品，下载量 135k，⭐ 650。**无需任何 API Key**，通过 `web_fetch` 直接聚合 16 个搜索引擎：国内 7 个（百度、必应CN、必应INT、360、搜狗、微信搜索、神马）+ 国际 9 个（Google、Google HK、DuckDuckGo、Yahoo、Startpage、Brave、Ecosia、Qwant、WolframAlpha）。自动检测查询语言，中文优先走国内引擎，英文走国际引擎，内置 1-2 秒限速和批次控制防封，失败时自动重试获取新 Cookie。结果跨引擎聚合去重输出。

**适用场景：**
- 同时检索国内（微信文章、搜狗新闻）和国际（Google、DuckDuckGo）源
- WolframAlpha 处理数学/科学计算型查询
- 无 API Key 环境下开箱即用，新部署机器零成本接入搜索
- 舆情监控：同时扫描微信公众号（wx.sogou.com）和国际新闻
- 与 Baidu Skill + Tavily Skill 互补，构建三层搜索策略

## 4. Nano Banana Pro（Gemini 3 Pro AI 图像生成与编辑）

**链接：** https://clawhub.ai/steipete/nano-banana-pro

**功能描述：**
@steipete 出品，下载量 93.3k，⭐ 372，今日刚更新。调用 Google Gemini 3 Pro Image API（即 Nano Banana Pro）实现文生图和图像编辑，支持 1K/2K/4K 三档分辨率。内置**草稿→迭代→终稿工作流**：先用 1K 快速验证 prompt，调整满意后再出 4K 高清图，避免在错误方向上消耗高分辨率额度。支持 `--input-image` 传入原图进行指令式编辑（修改构图、风格迁移、局部修改等）。需配置 `GEMINI_API_KEY`。

**适用场景：**
- 文字生成配图：一句话描述，输出高质量 4K 图
- 指令式图像编辑："把背景改成日落""删除右侧人物"
- UI 设计草图快速生成与迭代
- 内容创作者批量生成社交媒体配图
- 配合 Humanizer Skill：AI 写文章 → 生成配图 → 去 AI 味，全链路内容生产
