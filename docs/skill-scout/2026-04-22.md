# ⭐ 今日 Skill 推荐 — 2026-04-22

## 1. capability-evolver

**功能描述**

让 AI Agent 在运行过程中自主进化自身能力，无需人工干预。Agent 会在交互中持续学习、扩展行为边界，适合长期运行的自动化场景。

**适用场景**

- 需要长期运行、自我优化的 Agent
- 希望减少手动维护 prompt 和规则的团队
- 构建能随时间成长的个人助理

**安装**

```
clawhub install capability-evolver
```

**链接**：https://clawhub.ai/skills

## 2. self-improving-agent

**功能描述**

通过结构化日志（LEARNINGS.md / ERRORS.md / FEATURE_REQUESTS.md）记录错误、用户纠正和最佳实践，让 Agent 持续积累经验。支持将高价值经验提升到 AGENTS.md / SOUL.md 等工作区文件，实现跨会话知识传承。

**适用场景**

- 希望 Agent 记住自己踩过的坑
- 多 Agent 协作场景下共享经验
- 需要系统化管理 AI 学习记录的开发者

**安装**

```
clawhub install self-improving-agent
```

**链接**：https://clawhub.ai/pskoett/self-improving-agent

## 3. skill-finder

**功能描述**

智能 Skill 发现工具，同时搜索 ClawHub 和 Skills.sh 两个生态，根据用户需求推荐最合适的 Skill，并给出安装命令和对比分析。支持偏好记忆，避免重复推荐已排除的选项。

**适用场景**

- 不知道用哪个 Skill 完成某项任务
- 想对比 ClawHub 和 Skills.sh 的同类 Skill
- 希望 Agent 主动发现能力缺口并补全

**安装**

```
clawhub install skill-finder
```

**链接**：https://clawhub.ai/ivangdavila/skill-finder

## 4. felo-search

**功能描述**

专为 AI Agent 设计的搜索引擎，返回 AI 合成摘要 + 来源引用，而非原始链接列表。支持多语言，下载量超 145,000，是 ClawHub 第二热门 Skill。

**适用场景**

- Agent 需要快速获取结构化搜索结果
- 多语言内容研究
- 替代传统 web_search，减少 Agent 来回搜索轮次

**安装**

```
clawhub install felo-search
```

**链接**：https://clawhub.ai/skills

## 5. gog（Google Workspace CLI）

**功能描述**

将 Gmail、Calendar、Drive、Contacts、Sheets、Docs 统一为一个命令行接口。一句话完成"查本周日历冲突并起草改期邮件"这类跨工具操作，下载量约 14,000。

**适用场景**

- 深度使用 Google 生态的创业者和团队
- 希望用自然语言管理邮件 + 日历 + 文档
- 减少在多个 Google 产品间切换的上下文损耗

**安装**

```
clawhub install gog
```

**链接**：https://clawhub.ai/skills
