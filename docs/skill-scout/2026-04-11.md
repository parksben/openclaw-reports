# ⭐ 今日 Skill 推荐 — 2026-04-11

OpenClaw 生态中最热门的四大实用 Skills，帮助你提升工作效率。

## 1. **clawhub**
**功能描述：** ClawHub CLI 工具，用于搜索、安装、更新和发布 OpenClaw 代理技能。

**适用场景：**
- 快速发现和安装社区 skills
- 管理已安装 skills 版本更新
- 发布自己开发的 skills 到 ClawHub 社区
- 一键同步 skills 到最新版本

**优势：** 官方 npm 包，与 OpenClaw 核心深度集成，命令行友好。

---

## 2. **coding-agent**
**功能描述：** 将复杂编码任务委托给 Codex、Claude Code 或 Pi 代理，支持后台处理和迭代开发。

**适用场景：**
- 构建和创建新功能或应用
- 代码审查和 PR 处理
- 大型代码库重构
- 需要探索文件结构的迭代开发

**优势：** 支持多个代理平台，PTY 交互，文件感知上下文，适合复杂工程任务。

---

## 3. **feishu-doc**
**功能描述：** Feishu 文档读写操作，支持创建、编辑、表格操作和文件上传。

**适用场景：**
- 动态生成报告和汇总文档
- 建立文档和 OpenClaw 工作流的联动
- 团队协作文档自动化
- 集成表格数据处理

**优势：** 对接企业级协作平台，支持复杂表格操作，权限管理完善。

---

## 4. **gh-issues**
**功能描述：** GitHub 问题管理工具，支持获取 issues、生成修复 PR、监控审查评论。

**适用场景：**
- 自动化 bug 修复流程
- 批量处理 GitHub issues
- CI/CD 集成和 PR 自动跟踪
- 团队协作自动化

**优势：** 完整的 GitHub 工作流自动化，支持标签、里程碑、指派过滤。

---

## 5. **weather**
**功能描述：** 实时天气和预报查询，支持多个数据源（wttr.in、Open-Meteo）。

**适用场景：**
- 日程规划中的天气感知
- 出行前快速查询气象
- 集成到日常助手工作流
- 无需 API 密钥的轻量级查询

**优势：** 零配置、数据源多样、准确性高。
