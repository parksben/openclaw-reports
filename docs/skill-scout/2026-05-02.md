## 1. Self-Improving + Proactive Agent（自我进化 + 主动 Agent 合体版）

**链接：** https://clawhub.ai/ivangdavila/self-improving

**功能描述：**
下载量 178k，⭐ 1.1k，v1.2.16，22 小时前更新。将"自我进化"与"主动型 Agent"融为一体：自我反思（Self-reflection）+ 自我批评（Self-criticism）+ 自我学习（Self-learning）+ 自组织记忆（Self-organizing memory）。采用三层记忆架构（HOT/WARM/COLD），通过 `~/self-improving/` 目录持久化存储纠错记录、偏好规则、项目模式。每次被用户纠正或完成重要任务后自动记录并学习，支持 heartbeat 定期维护，与 Proactive Agent 深度整合。

**适用场景：**
- Agent 频繁犯同类错误，希望真正"记住"纠正
- 多 session 长期使用，需跨会话积累个人偏好与工作风格
- 构建自我迭代的开发/写作/运营 Agent，让 AI 越用越懂你
- 配合 heartbeat cron 实现定期记忆维护与归档

## 2. SkillScan（Skill 安全扫描门卫）

**链接：** https://clawhub.ai/tokauthai/skillscan

**功能描述：**
下载量 166k，⭐ 12，1 天前更新。定位为每个新 Skill 安装前的"强制安全门"，与 Skill Vetter 不同之处在于：它将安全扫描定义为工作流中**不可跳过**的前置步骤，任何安装/加载/评估 Skill 的动作都自动触发。执行 VirusTotal + 静态代码分析 + ClawScan 三重检测，输出带风险等级的结构化扫描报告。

**适用场景：**
- 团队统一安全准入流程，禁止未扫描 Skill 上线
- 评估 ClawHub、GitHub 等来源的第三方 Skill 安全性
- 识别 curl/wget 外传、凭据读取、eval 执行等危险模式
- 作为 auto-updater 更新前的安全前置校验

## 3. Tavily 搜索（Brave 不可用时的网络搜索替代）

**链接：** https://clawhub.ai/jacky1n7/openclaw-tavily-search

**功能描述：**
下载量 88.7k，⭐ 236，21 小时前更新。通过 Tavily API 提供高质量网络搜索能力，作为 Brave Search 不可用时的替代方案。支持关键词搜索、返回原文链接与摘要，搜索结果质量经过 AI 优化筛选，比原始 Google 搜索更聚焦于答案。需要配置 Tavily API Key（免费额度足够日常使用）。

**适用场景：**
- Brave 搜索不可用或额度耗尽时无缝切换
- 需要高质量 AI 优化搜索结果（而非原始爬虫列表）
- 研究型 Agent 工作流中获取可信来源与摘要
- 与 multi-search-engine 组合构建多源搜索策略

## 4. Baidu Web Search（百度 AI 搜索引擎）

**链接：** https://clawhub.ai/ide-rea/baidu-search

**功能描述：**
下载量 82.5k，⭐ 208，33 分钟前刚更新。调用百度 AI 搜索引擎（BDSE）进行实时网络搜索，适合中文内容查询、国内资讯获取、文档与技术资料检索。相比通用搜索引擎在中文语境下覆盖更完整，结合 AI 理解层返回更精准的摘要。

**适用场景：**
- 中文内容、国内新闻、政策法规的实时查询
- 访问国内技术文档、社区（CSDN、知乎、博客园）内容
- 需要国内特定信息时替代 Google/Bing
- 配合 Multi Search Engine 构建中英文双轨搜索策略
