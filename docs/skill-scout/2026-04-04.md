# ⭐ Skill 推荐 — 2026-04-04

## 1. gh-issues
- **功能：** 自动化 GitHub issue 管理，支持批量获取、自动修复和 PR 监控
- **适用场景：** 开发者快速处理 GitHub bugs、自动生成修复 PR、追踪 PR review 评论
- **链接：** [clawhub.ai/gh-issues](https://clawhub.ai/skills/gh-issues)

## 2. notion
- **功能：** 完整 Notion API 集成，支持创建/管理页面、数据库和内容块
- **适用场景：** 与 Notion 工作区同步，自动化笔记管理、数据库更新、内容发布
- **链接：** [clawhub.ai/notion](https://clawhub.ai/skills/notion)

## 3. weather
- **功能：** 实时天气查询和预报（支持全球任何地点），无需 API key
- **适用场景：** 智能助手的日常检查列表、活动规划、天气相关提醒
- **链接：** [clawhub.ai/weather](https://clawhub.ai/skills/weather)

## 4. skill-creator
- **功能：** 创建、编辑、改进和审计 Agent Skills，自动化 skill 开发工作流
- **适用场景：** 为 OpenClaw 社区贡献新 skills、维护现有 skill、代码审计和重构
- **链接：** [clawhub.ai/skill-creator](https://clawhub.ai/skills/skill-creator)

## 5. node-connect
- **功能：** 诊断和修复 OpenClaw 节点连接失败（支持 Android/iOS/macOS）
- **适用场景：** 排查配对设备失败、本地网络与 VPS 连接问题、Tailscale 配置调试
- **链接：** [clawhub.ai/node-connect](https://clawhub.ai/skills/node-connect)
