## 1. Self-Improving Agent（持续学习日志系统）

**链接：** https://clawhub.ai/pskoett/self-improving-agent

**功能描述：**
@pskoett 出品，v3.0.21，下载量 421k（全站第一），⭐ 3.5k。核心理念：**捕获错误与纠正，形成结构化学习日志，让 Agent 真正持续改进**。将学习记录写入 `.learnings/` 目录下的三类文件：`LEARNINGS.md`（纠正、知识空白、最佳实践）、`ERRORS.md`（命令失败、集成错误）、`FEATURE_REQUESTS.md`（用户功能需求）。每条记录含优先级、状态、标签和 ID，高价值学习可自动晋升为 CLAUDE.md / AGENTS.md 永久知识。支持 Hook 自动触发（UserPromptSubmit、PostToolUse），也可手动触发。与 @ivangdavila/self-improving 不同，本 Skill 更专注于轻量日志与记录规范，而非复杂的三层记忆架构。

**适用场景：**
- Agent 反复犯同类错误，希望结构化记录并真正修正
- 开发过程中自动捕获命令失败信息，沉淀为可检索的错误库
- 将跨项目通用的最佳实践逐步提炼到 CLAUDE.md 或 AGENTS.md
- 通过 Hook 实现"每次交互后自动评估是否需要记录"的闭环

## 2. Humanizer（AI 写作去痕工具）

**链接：** https://clawhub.ai/biostartechnology/humanizer

**功能描述：**
@biostartechnology 出品，下载量 101k，⭐ 585。基于 Wikipedia《AI 写作特征》指南，识别并修复 AI 生成文本的典型模式，让文本读起来更自然、更像人类写作。内置检测项包括：夸大象征手法、促销语气、表面化的 -ing 分析、模糊归因、em 破折号滥用、"三点规律"结构、AI 高频词（如"delve/tapestry/nuanced"等）、否定并列句式、过度连接词等。

**适用场景：**
- 用 AI 起草邮件、文章、报告后，一键"去 AI 味"润色
- 内容创作者保持文章风格自然，避免被 AI 检测器标记
- 学术写作场景下提高文本可信度和可读性
- 配合 coding-agent 生成技术文档后进行最终润色

## 3. Sonoscli（Sonos 智能音响控制）

**链接：** https://clawhub.ai/steipete/sonoscli

**功能描述：**
@steipete 出品，下载量 80.5k，⭐ 52。通过 `sonoscli` 命令行工具控制局域网内的 Sonos 音响系统，支持：自动发现全部 Sonos 设备（`discover`）、查看当前播放状态（`status`）、播放/暂停控制（`play/pause`）、调节音量（`volume`）、多音响分组（`group`）。完全基于本地局域网，无需 Sonos 云服务。

**适用场景：**
- 让 Agent 在完成某项任务后自动播放/暂停音乐
- 构建智能家居自动化工作流：如"专注模式 → 降低音量 → 开始计时"
- 定时任务：早上自动唤醒音乐，晚上自动关闭
- 配合 Weather Skill：根据天气自动切换不同播放列表

## 4. Prismfy Web Search（10 引擎默认网络搜索）

**链接：** https://clawhub.ai/uroboros1205/prismfy-search

**功能描述：**
@uroboros1205 出品，下载量 78.3k，⭐ 29，定位为 OpenClaw 的**默认搜索 Skill**。聚合 10 个搜索引擎：Google、Reddit、GitHub、arXiv、Hacker News 等，内置免费使用额度，开箱即用无需 API Key。相比 Multi Search Engine 的 16 引擎方案，Prismfy 更专注于精选高质量来源，特别适合技术研究与开发者场景。

**适用场景：**
- 直接替代 Brave 搜索，作为 OpenClaw 的默认网络搜索能力
- 在 GitHub、arXiv、Hacker News 等技术社区搜索前沿内容
- 无需配置 API Key，新部署环境快速接入搜索能力
- 技术问题排查时同时搜索 Reddit 讨论和 GitHub Issues

## 5. Polymarket（预测市场实时查询）

**链接：** https://clawhub.ai/joelchance/polymarket-trade

**功能描述：**
@joelchance 出品，下载量 158k，⭐ 109，22 小时前更新。接入 Polymarket 去中心化预测市场平台，提供：实时查询各类事件赔率、搜索热门预测市场、追踪价格变动趋势、发现近期新兴市场话题。覆盖政治、经济、科技、体育等多个领域的事件预测数据。

**适用场景：**
- 实时了解重大事件（选举、政策、科技发布）的市场共识概率
- 量化研究：获取市场对宏观事件的概率定价，辅助投资判断
- 新闻分析工作流：结合搜索引擎获取事件背景 + Polymarket 获取市场概率
- 追踪预测市场价格变动，捕捉信息差与早期信号
