## 1. OpenClaw Token Optimizer — Token 用量优化

**作者：** ccjingeth | **版本：** 1.0.0  
**链接：** https://clawhub.ai/ccjingeth/openclaw-token-save

### 功能描述

专为 OpenClaw 设计的 Token 成本优化 Skill。通过审计上下文注入、裁剪工作区文件（AGENTS.md / SOUL.md / MEMORY.md 及每日记忆日志）、启用提示词压缩等手段，系统性降低每轮对话的 Token 消耗。

### 适用场景

- Token 账单超出预算，需要找到"大户"进行瘦身
- 工作区文件越来越臃肿，影响响应速度
- 希望在不损失质量的前提下降低 API 调用成本

## 2. Home Assistant Controller — 智能家居控制

**作者：** chris6970barbarian-hue | **版本：** 1.0.0  
**链接：** https://clawhub.ai/chris6970barbarian-hue/glitch-homeassistant

### 功能描述

通过 Home Assistant API 控制和监控智能家居设备的 Skill。支持灯光、开关、窗帘、温控器、场景和脚本的命令式操作，只需一句话即可调度家中所有接入 HA 的设备。

### 适用场景

- 用自然语言控制智能家居（"把客厅灯调到 50% 暖白"）
- 查询当前设备状态（温湿度、开关状态）
- 触发 HA 场景模式（回家模式、睡眠模式）

## 3. Web Form Automation — 网页表单自动化

**作者：** flyingzl | **版本：** 1.0.0  
**链接：** https://clawhub.ai/flyingzl/web-form-automation

### 功能描述

基于 Playwright 的网页表单自动化 Skill，支持登录、文件上传、文本输入和表单提交等完整交互流程。适合需要与有登录墙、验证码或复杂交互的网站打交道的自动化场景。

### 适用场景

- 自动化填写重复性表单（报销、申请、注册）
- 模拟用户登录流程后抓取需认证的内容
- 批量文件上传或数据提交任务

## 4. LLM Council Router — 智能 LLM 路由

**作者：** ashtiwariasu | **版本：** 1.0.0  
**链接：** https://clawhub.ai/ashtiwariasu/llmcouncil-router

### 功能描述

基于 LLM Council 同行评审排名，将提示词自动路由到当前最优 LLM 的 Skill。无需手动切换模型，系统根据任务类型和最新基准数据选出最合适的模型来执行。

### 适用场景

- 希望每次任务都用到"最强模型"而无需手动判断
- 多模型对比测试，自动选最优结果
- 对模型性能敏感的生产场景（代码生成、推理、写作）

## 5. OpenClaw Migrate — 一键迁移 OpenClaw

**作者：** chris6970barbarian-hue | **版本：** 1.0.0  
**链接：** https://clawhub.ai/chris6970barbarian-hue/openclaw-migrate

### 功能描述

通过 SSH 将 OpenClaw 的全套配置迁移到新主机的 Skill。涵盖配置文件、Skills、记忆文件、Token、环境变量和 Cron 任务，支持 setup / test / migrate 三步流程，确保迁移前后行为一致。

### 适用场景

- 换服务器或 VPS，不想手动重配 OpenClaw
- 备份当前 Agent 状态到新机器
- 多机器环境同步 OpenClaw 配置
