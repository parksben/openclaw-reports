## 1. ontology（类型化知识图谱）

**链接：** https://clawhub.ai/oswalpalash/ontology

**功能描述：**
为 Agent 提供结构化知识图谱存储能力。支持定义 Person、Project、Task、Event、Document 等实体类型，通过类型约束+关系图谱管理 Agent 的长期记忆与跨技能共享状态。采用追加写入（Append-Only）模式保证历史可溯，支持 Python CLI 进行 create/query/relate/validate 操作。v1.0.4，下载量 173k，⭐ 571。

**适用场景：**
- Agent 需要跨 session 持久化结构化记忆（任务、人物、项目关系）
- 多 Skill 协作时共享上下文与状态
- 构建复杂多步骤工作流，以图变换方式建模计划执行

## 2. Gog（Google 全家桶 CLI）

**链接：** https://clawhub.ai/steipete/gog

**功能描述：**
一站式 Google Workspace CLI，覆盖 Gmail、Google Calendar、Google Drive、Contacts、Sheets 和 Docs。无需分别配置多个工具，用统一命令行接口操控 Google 全套服务。下载量 164k，⭐ 853。

**适用场景：**
- 读写 Gmail 邮件、自动回复、邮件归档
- 创建/修改 Google Docs / Sheets
- 管理 Google Calendar 日程与 Drive 文件

## 3. Polymarket（预测市场查询）

**链接：** https://clawhub.ai/joelchance/polymarket-trade

**功能描述：**
接入 Polymarket 预测市场平台，实时查询各类事件的赔率、搜索热门预测市场、追踪价格走势。适合对宏观事件概率感兴趣的用户或量化研究场景。下载量 143k，⭐ 99。

**适用场景：**
- 实时了解政治、经济、科技等大事件的市场预测概率
- 筛选热门或近期波动大的预测市场
- 量化研究或投资辅助决策

## 4. Multi Search Engine（16 引擎聚合搜索）

**链接：** https://clawhub.ai/gpyangyoujun/multi-search-engine

**功能描述：**
聚合 16 个搜索引擎（7 个国内 + 9 个国际），包括百度、必应、Google、DuckDuckGo、Wolfram Alpha 等，支持高级搜索运算符、时间过滤、站内搜索、隐私引擎切换。下载量 130k，⭐ 610，2 周前更新。

**适用场景：**
- 需要跨引擎对比搜索结果，避免信息茧房
- 访问国内/国际不同来源的搜索结果
- 搜索特定时间范围或特定站点的内容

## 5. Nano Banana Pro（Gemini 图像生成）

**链接：** https://clawhub.ai/steipete/nano-banana-pro

**功能描述：**
基于 Gemini 3 Pro Image 模型的图像生成与编辑 Skill。支持文字生图（Text-to-Image）和图片编辑（Image-to-Image），分辨率可选 1K/2K/4K，通过 `--input-image` 参数传入参考图。今日（2026-04-28）刚更新，下载量 91.1k，⭐ 360。

**适用场景：**
- 根据文字描述快速生成高质量图片
- 对现有图片进行 AI 风格改写或内容编辑
- 设计原型、插图生成、社交媒体视觉内容创作
