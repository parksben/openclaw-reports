## 1. self-improving-agent（自我进化 Agent）

**链接：** https://clawhub.ai/pskoett/self-improving-agent

**功能描述：**
捕获 Agent 的报错、用户纠正、操作失败等事件，持续积累经验教训，让 Agent 随着使用次数增加变得越来越聪明。v3.0.18，2026-04-25 刚更新，活跃维护中。

**适用场景：**
- Agent 反复犯同类错误，需要长期纠偏
- 让 AI 助手记住你的纠正习惯，减少重复提醒
- 构建具有自我改进能力的 Agent 工作流

## 2. elite-longterm-memory（终极长期记忆系统）

**链接：** https://clawhub.ai/nextfrontierbuilds/elite-longterm-memory

**功能描述：**
为 Cursor、Claude、ChatGPT、Copilot 等各类 AI Agent 提供统一的长期记忆方案。采用 WAL 协议（先写后答）+ 向量搜索 + git-notes + 云备份，彻底解决上下文压缩导致的记忆丢失问题。v1.2.3，今日（2026-04-27）更新。

**适用场景：**
- 长对话、多会话需要跨 session 持久记忆
- 多 Agent 协作共享知识库
- Vibe Coding 场景下保持项目上下文连续性

## 3. pdf（PDF 全功能处理工具箱）

**链接：** https://clawhub.ai/awspace/pdf

**功能描述：**
全面的 PDF 操作工具集，支持文字和表格提取、新建 PDF、合并/拆分文档、填写表单等。适合需要批量处理或编程操作 PDF 文档的场景。今日（2026-04-27）更新至 v0.1.0。

**适用场景：**
- 提取合同、报告中的结构化数据
- 批量合并或拆分 PDF 文件
- 自动填写 PDF 表单

## 4. humanizer（AI 写作去痕工具）

**链接：** https://clawhub.ai/biostartechnology/humanizer

**功能描述：**
基于 Wikipedia「AI 写作特征」指南，识别并修复 AI 生成文本中的典型痕迹：夸张象征、推广性语言、em dash 滥用、三段式结构、AI 高频词汇、负面排比句等，让输出读起来更自然、更像人写的。

**适用场景：**
- 润色 AI 辅助撰写的文章、邮件、报告
- 内容创作者提升输出可读性
- 避免 AI 检测工具误报

## 5. security-auditor（代码安全审计专家）

**链接：** https://clawhub.ai/jgarrison929/security-auditor

**功能描述：**
专注代码安全审查的 Skill，覆盖 OWASP Top 10 漏洞、认证流程实现、CORS/CSP 配置、Secrets 管理、输入校验、SQL 注入防护、XSS 防护等安全关键场景。适合开发者在 PR Review 或上线前进行安全检查。

**适用场景：**
- 上线前代码安全全面扫描
- 审查 API 接口的认证与授权逻辑
- 排查 Web 应用常见安全漏洞
