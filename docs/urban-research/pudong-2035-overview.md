## 三、四大核心板块详解

### 🟡 世博-前滩 — 新晋城市主中心

2024年3月正式确立为浦东第二个 **城市主中心（CAZ）**，与陆家嘴并列。

| 指标 | 数据 |
|------|------|
| 总规划面积 | 约 35 km² |
| 已建成商办 | 360 万㎡ |
| **待开发商办** | **240 万㎡（最大增量）** |
| 已入驻央企 | 28栋，含商飞、宝钢、国家电网、中信等 |
| 文化配套 | 上海大歌剧院、世博文化公园（2km²） |
| 在建地铁 | 19号线（串联陆家嘴-前滩-张江） |

> 📌 **240万㎡待开发商办 = 全市最大单片区在建商办增量**，是2025-2030年土地出让最活跃板块之一。

---

### 🟣 张江科学城 — 科创策源引擎

| 指标 | 数据 |
|------|------|
| 大张江全域面积 | 220 km² |
| 核心建设区 | 约 60 km² |
| **新增住宅总量** | **920 万㎡** |
| 其中租赁住宅 | **890 万㎡（97%）** |
| 商品房供应 | 约 30 万㎡（极度稀缺） |
| 重大科学装置 | 12个国家级基础设施 |
| 产业规模 | 生物医药 2025年突破千亿 |

> 📌 张江商品房稀缺 → **大量就业人口需求外溢至三林、北蔡、唐镇**，是推高周边板块房价的核心引擎。
