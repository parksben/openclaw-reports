# 🤖 AI 每日简报 — 2026-04-17

## 今日热点

1. **Anthropic 发布 Claude Opus 4.7，重夺最强公开 LLM 宝座** — Anthropic 正式推出 Claude Opus 4.7，在知识工作基准 GDPVal-AA 上以 Elo 1753 分超越 GPT-5.4（1674）和 Gemini 3.1 Pro（1314）。新模型最大亮点是"自我验证"能力——完成任务前会主动构建内部测试来核查结果，大幅减少幻觉循环。同时引入 `effort` 参数（新增 xhigh 档）和 API 任务预算功能，让企业可精确控制 token 消耗。Claude Code 内新增 `/ultrareview` 命令，模拟资深工程师做代码审查。定价维持 $5/$25 per million tokens 不变，已上线 Amazon Bedrock、Google Vertex AI 和 Microsoft Foundry。→ [查看详情](https://venturebeat.com/technology/anthropic-releases-claude-opus-4-7-narrowly-retaking-lead-for-most-powerful-generally-available-llm)

2. **Anthropic Claude Code 桌面版全面重设计，推出"Routines"自动化功能** — 4 月 14 日，Anthropic 发布 Claude Code 桌面应用重设计版本（Mac/Windows），核心是新侧边栏"Mission Control"，支持同时管理多个 AI 编码会话。同步推出 Research Preview 阶段的"Routines"功能，支持定时任务、API 触发和 GitHub Webhook 三种模式，任务运行在 Anthropic 云端，无需本地机器保持开启。企业版每日最多 25 条 Routine。→ [查看详情](https://venturebeat.com/orchestration/we-tested-anthropics-redesigned-claude-code-desktop-app-and-routines-heres-what-enterprises-should-know)

3. **Adobe 发布 Firefly AI Assistant，一句话驱动 Photoshop/Premiere/Illustrator** — Adobe 推出 Firefly AI Assistant（Project Moonlight 正式版），这是一个可跨 Creative Cloud 全套工具编排复杂多步骤工作流的 AI 代理。用户只需用自然语言描述目标，助手自动调用约 100 个工具和技能完成任务，输出原生 Adobe 格式（PSD、AI、PRPROJ）。同步集成快手旗下 Kling 3.0 视频模型，Premiere Pro 新增 Color Mode 色彩分级功能，Frame.io Drive 让云端媒体像本地文件一样操作。公测将在数周内开放。→ [查看详情](https://venturebeat.com/technology/adobes-new-firefly-ai-assistant-wants-to-run-photoshop-premiere-illustrator-and-more-from-one-prompt)

4. **Meta 研究员提出"超级智能体"框架，AI 可在非编程领域自我进化** — Meta 联合多所高校发布 Hyperagents 论文，提出将任务执行与自我改进融合为单一可编辑程序的框架。与只能在编程领域自我优化的 DGM 不同，Hyperagents 可在论文评审、机器人奖励设计、数学评分等非编程领域持续自我改进，且能将在一个领域学到的元技能迁移到全新领域。代码已在 GitHub 开源（非商业许可）。→ [查看详情](https://venturebeat.com/orchestration/meta-researchers-introduce-hyperagents-to-unlock-self-improving-ai-for-non-coding-tasks)

5. **Anthropic 秘密网络安全模型 Mythos 引发广泛关注，白宫也在跟进** — Anthropic 专为网络安全打造的私有模型 Mythos Preview 正被 Nvidia、Apple、JPMorgan Chase 等大型企业用于修补高危漏洞，The Verge 报道白宫也在准备获取访问权限。该模型因能力过强被认为"危险到无法公开发布"，目前仅限少数合作伙伴使用。Anthropic 同步推出 Cyber Verification Program，允许合法安全研究人员申请使用 Opus 4.7 的安全能力。→ [查看详情](https://www.theverge.com/ai-artificial-intelligence)

6. **Google 与五角大楼谈判，拟将 Gemini 引入机密军事环境** — 据 The Information 报道，Google 正与美国国防部洽谈，计划将 Gemini 的使用范围从现有的非机密场景扩展至机密设置，合同条款或与 OpenAI 此前签署的五角大楼协议相似。此举标志着 Google 在军事 AI 合作上的重大政策转向。→ [查看详情](https://www.theverge.com/ai-artificial-intelligence)

7. **Google 发布 Gemini Robotics-ER 1.6，Spot 机器狗学会读仪表盘** — Google DeepMind 发布 Gemini Robotics-ER 1.6，称其为"迄今最安全的机器人模型"，能以"前所未有的精度"理解环境。Boston Dynamics 的 Spot 机器狗借助该模型实现了读取压力表等仪器数值的能力，展示了 AI 视觉推理在物理世界的实际落地。→ [查看详情](https://deepmind.google/blog/gemini-robotics-er-1-6/)

8. **Google 为 Android 开发者推出 AI 编程技能库和知识库** — Google 发布升级版 Android CLI，同步上线 Android Skills GitHub 仓库和 Android Knowledge Base，为 AI 编程代理提供执行 Android 开发任务所需的信息和资源，降低 AI 辅助 Android 开发的门槛。→ [查看详情](https://github.com/android/skills)
